# Markdown转PDF应用开发规划文档

## 一、项目概述

本项目旨在开发一套基于Rust + Actix Web后端、React 19 + Tailwind CSS前端技术栈的Markdown转PDF本地渲染程序，支持Mermaid图表渲染，具备安全性、可扩展性和良好的用户体验。系统采用Tauri 2.0进行跨平台打包分发，确保在Windows、Mac OS、Linux等主流操作系统下均可流畅运行。

## 二、技术选型

### （一）前端技术栈

1. **核心框架**：
   - React 19
   - TypeScript 5.0
   - Vite 5.0

2. **UI框架**：
   - Tailwind CSS 4
   - Headless UI
   - React Icons

3. **编辑器**：
   - Monaco Editor

4. **图表渲染**：
   - Mermaid.js 10.0

### （二）后端技术栈

1. **核心框架**：
   - Rust 1.75
   - Actix Web 4.4
   - Tokio 1.35

2. **PDF生成**：
   - WeasyPrint

3. **文件处理**：
   - Rust标准库
   - Serde

4. **日志系统**：
   - Log4rs
